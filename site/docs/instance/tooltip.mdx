# tooltip

If the field defined, tooltips will be displayed instead of regular error labels. It has position field which could be 'left' | 'top' | 'right' | 'bottom'.

### Type

`{ position: 'left' | 'top' | 'right' | 'bottom' }`

### Value by default

`undefined`

### Example

```js
{
  tooltip: {
      position: 'top',
  },
}
```
